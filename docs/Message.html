Class: Message

Message

new Message(client [, data])

This class represents a message sent in any slack channel.

Parameters:
Name Type Argument Description
client Client

The Slack Client to use when sending messages and pulling data.
data object <optional>

The 'raw' data provided by slack if applicable.
