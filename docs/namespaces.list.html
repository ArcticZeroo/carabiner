message

Emitted when a message is sent in a slack channel. These are NOT necessarily chat messages! Listen to message.chat event instead for those. If you just want to listen to a single subtype, you can listen to message.subtype.. Or if you only want messages from a certain channel type, you can listen to message.type.

Parameters:
Name Type Description
message Message

The message that was sent.
