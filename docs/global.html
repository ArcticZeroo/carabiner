createSlackRequestUrl(method [, query])

Create a slack request URL based on a query object and a method path. This method does basic stringifying of the @query param. If a value is not a string, it's stringified with JSON.stringify. Then, the string (no matter if it started as a string or not) is encoded with encodeURIComponent. This is because slack has a hard time understanding directly encoded 'objects' like {hello: 'world'}. Stringification before encoding works well enough. It's possible that this may cause some issues for some specific characters, but so far I have encountered none, even with emoji.
Parameters:
Name Type Argument Default Description
method string The method to request.
query object <optional>
 {} The query object to stringify.
Returns:
Type
string

deprecate(fn, msg)

Mark that a method should not be used. Returns a modified function which warns once by default. If `localStorage.noDeprecation = true` is set, then it is a no-op. If `localStorage.throwDeprecation = true` is set, then deprecated functions will throw an Error when invoked. If `localStorage.traceDeprecation = true` is set, then deprecated functions will invoke `console.trace()` instead of `console.error()`.
Parameters:
Name Type Description
fn function the function to deprecate
msg String the string to print to the console when `fn` is invoked
Returns:
a new "deprecated" version of `fn`
Type
function
