Class: Attachment

Attachment

new Attachment()

Create a slack attachment object. No build, json-ifying or any hocus-pocus necessary. All of the properties on this object are the ones slack needs to see, so JSON.stringify makes it valid to send. All methods are chainable. It's essentially a builder, but you don't need to ever build it.

Example
new Attachment()
 .setTitle('hello')
 .setText('world!')
 .setColor('#2196F3)
 .addField(new Field()...)
