Class: Client

Client

new Client(token [, opts])

Create a single-team Slack Client.
Parameters:
Name Type Argument Default Description
token string The slack token to use for all operations.
opts object <optional>
 {} Options to use.
Properties
Name Type Argument Default Description
rtm boolean <optional>
 true Whether the client should connect to RTM.
getFullChannelInfo boolean <optional>
 false Whether the client should retrieve ALL info about channels. This will significantly increase startup time if true, and could hit rate limit.
separateGroupAndMpim boolean <optional>
 false Whether group and mpim channels should have a different type.
getUserPresence boolean <optional>
 false Whether user preference should be added to cached data. This will significantly increase startup time if true.
useRtmStart boolean <optional>
 false Whether to use rtm.start for all init info rather than rtm.connect. If this is false, each thing is separately retrieved by the web api
